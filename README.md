#Projeto para Consultoria#

Status do projeto: Em desenvolvimento...

Tecnologias aplicadas: MySQL, Apache NetBeans e Linguagem de Programação Java.

Time de Desenvolvedores: Apenas eu

Objetivo do software: Esse software foi feito para construir um projeto para uma determinada consultoria, com objetivo de criar algo do zero e ir implementando as funcionalidades necessárias no futuro.

Funcionalidades do Sistema:

#1. Cadastro de Clientes:#

O sistema deve permitir o cadastro de novos clientes.

Campos obrigatórios: nome completo, RG, CPF, telefone e email.

O sistema deve validar se o CPF informado é válido.

O cliente deve ser cadastrado em uma base de dados e ser acessível para edições futuras.

#2. Cadastro de Consultorias:#

O sistema deve permitir o cadastro de novos serviços de consultoria.

Campos obrigatórios: nome da consultoria, descrição, preço e duração.

O sistema deve permitir a atualização de consultorias já cadastradas.

#3. Agendamento de Consultorias:#

O sistema deve permitir o agendamento de consultorias pelos clientes.

O usuário deve escolher o cliente, consultoria e horário disponível.

O sistema deve verificar a disponibilidade do horário antes de realizar o agendamento.

#4. Inserção de Funcionarios:#

O sistema pede informações relevantes de cada funcionario como: seu nome, sua área de atuação e etc...

Apenas administradores podem excluir os funcionarios.

O sistema deve listar funcionarios cadastrados.

#5. Login de Usuário:#

O sistema deve permitir o login de administradores e funcionarios dependendo do seu tipo de objetivo.

O login deve ser feito através de email e senha.

O sistema deve garantir a segurança das senhas, utilizando técnicas de criptografia.
